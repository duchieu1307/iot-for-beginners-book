= Cheatsheet

This article provides information for quickly tracking functions that can be used with Arduino, ESP8266 and the C programming language. There are many libraries for Arduino which provides useful APIs, this brief content only mention about the most popular libraries. If you use another library, you can easily check its user guide. Normally, opened source library on https://github.com[Github] included the README.md file, which contains all the information.
