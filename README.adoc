== How To Generate the Book

You can generate the e-book files manually with Asciidoctor. If you run the following you may actually get HTML, (Epub, Mobi - need to fix images dir) and PDF output files:

----
$ bundle install
$ LANG=en make pdf
$ LANG=vi make pdf
$ LANG=en make html
$ LANG=vi make html
----
